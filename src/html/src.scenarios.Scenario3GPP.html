def __generate_LOSgrid(self, corr_distance):

This method generates a spatial correlated uniform rvs for LOS.

The covariance matrix is defined by cov = exp(-distance/correlation_distance). Copula method is used: first it generates gaussian rvs with the desired correlation; second, using gaussian cdf it converts the rvs to uniform[0,1] rvs (this is the last step in our case). If it is necessary rvs with other distribution, using the inverse distribution can be generated the rvs with the desired distribution from the uniform rvs.

Parameters
corr​_distance:float.The correlation distance for the spatial correlated gaussian random variable.
Returns
A 2D matrix where the values of the matrix are spatially correlated uniform random variables.
